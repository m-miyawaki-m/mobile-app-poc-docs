# mobile-app-poc docs

`mobile-app-poc` プロジェクトの **設計文書・調査メモ・ワークフロー解説** を集約するリポジトリ。
コードは別リポ（`mobile-app-poc-api-spec` / `-mock` / `-frontend` / `-bridge` / `-backend`）にある。

## 構成

| ファイル | 内容 |
|---|---|
| `1. mobile-app-poc-全体統括.md` | プロジェクト原典 — 全体方針・マルチリポ構成 |
| `2. mobile-app-poc-api-spec.md` | api-spec リポの原典 |
| `3. mobile-app-poc-mock.md` | mock リポの原典 |
| `4. mobile-app-poc-frontend.md` | frontend リポの原典 |
| `5. mobile-app-poc-bridge.md` | bridge リポの原典 |
| `6. mobile-app-poc-backend.md` | backend リポの原典 |
| `ionic-vue-workflow.md` | Ionic Vue 開発フロー総覧 + VS Code/Kotlin 環境構築 |
| `superpowers/specs/` | 各リポの詳細設計書 |
| `superpowers/plans/` | 各リポの実装計画 |

## 関連リポジトリ

## 読み始める順序

1. `1. mobile-app-poc-全体統括.md` でプロジェクト全体像を把握
2. `superpowers/specs/2026-04-19-mobile-app-poc-master-design.md` で構築戦略を把握
3. `ionic-vue-workflow.md` で開発フロー・環境構築を把握
4. 各リポの README で具体操作を確認
